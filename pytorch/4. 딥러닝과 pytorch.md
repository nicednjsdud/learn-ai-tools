# 딥러닝과 pytorch

# PyTorch(파이토치), Keras(케라스), TensorFlow(텐서플로우), Theano(티아노)

## Keras

* 파이썬으로 구현된 간결한 딥러닝 라이브러리
* 딥러닝 구현을 쉽게 할 수 있도록 하는 **상위 레벨의 인터페이스를 제공**함
* 내부적으로 TensorFlow, Theano 등의 딥러닝 전용 엔징을 사용할 수 있음
* 장점 : 직관적
* 단점 : 상위 레벨의 인터페이스이므로, 복잡한 프로젝트에서는 구현 범위에 한계가 있을 수 있음

## Pytorch

* 페이스북 AI 연구팀이 개발한 파이썬 기반 딥러닝 라이브러리
* Keras 와 마찬가지로 직관적인 인터페이스 제공
* **TensorFlow는 정적으로 그래프를 생성하지만, Pytorch는 동적으로 그래프를 생성**
    * pyTorch는 메모리에서 연산을 수행하면서, 그래프(신경망) 사이즈를 최적으로 바꾸며 동작시키므로, 성능이 더 나을 수 있음
* 러닝커브가 낮은 편임

## TensorFlow

* Theano를 기반으로, 구글에서 개발
* 기존에 많은 코드가 TensorFlow 기반이므로, 관련 예제 코드등이 보다 많은 편임
* 단점은 러닝커브가 심함
