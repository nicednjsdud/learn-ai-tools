
# 인공 신경망 (Artificial Neural Network)

* 인공 신경망도 여러 타입 및 용어가 있음

## 피드 포워드 신경망 (Feed-Forward Neural Network, FFNN)

* 오직 입력층에서 출력층 방향으로 연산이 전개되는 다층 퍼셉트론을 의미함

## 순환 신경망 (Recurrent Neural Network, RNN)

* FFNN 과는 달리, 입력층 또는 은닉층의 연산이 출력층으로만 전개되지 않고, 또 다른 은닉층등의 입력으로도 사용되는 다층 퍼셉트론 구조

## Linear Layer (선형 계층)

* 모든 노드가 연결되어 있는 Layer를 Full-Connected Layer 라고도 함
* 전결합층 (Full-Connected Layer), FC Layer, 또는 Dense Layer 라고도 함
* 모든 입력 노드가, 출력 노드에 연결되어 있는 형태로, DNN의 가장 기본이 되는 구성 요소
* $y = x * W + b$을 기반으로 선형 관계를 근사함
* 결국, 입력 차원을 특정 차원으로 변형하는 역할을 함

  * 예 : $f : \mathbb{R}^4 -> \mathbb{R}^3$ 

* Linear Layer는 딥러닝에서 행렬로 연산됨 (이후 코드와 함께 구현레벨에서 선명하게 이해하기로 함)
