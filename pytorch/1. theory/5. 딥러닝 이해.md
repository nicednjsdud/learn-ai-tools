
# 딥러닝 이해

## 퍼셉트론(perceptron)

### 뉴런의 이해

* 신경세포 1개를 의미하며, 인간의 뇌는 약 1000억개의 뉴런으로 구성됨
* 뉴런은 크게 세가지 부분으로 나눌 수 있음

  * 신경세포체 : 핵이 있는 세포 부분
  * 가지돌기 : 다른 세포에서 신호를 받는 부분
  * 축삭돌기 : 다른 세포에 신호를 주는 부분
  * 시냅스 : 돌기 사이에 다른 뉴런에 신호를 전달하는 부분

* 가지돌기에 자극을 받게 되면, 신경세포체를 통해 축삭돌기에 화학물질이 전달되어, 전이가 발생하고,
* 전이의 정도가 특정 임계값을 넘으면, 시냅스를 통해, 신호를 다음 뉴런에게 전달하는 구조

### 인공 신경망 (artificial neural network, ANN)

* 웨렌 맥컬록과 윌터 피츠가 1943년도 논문 발표

  * 뉴런의 작용을 0과 1로 이루어진 이진법 논리 회로 모델로 구성
  * 이런 모델을 그물망처럼 연겨하면, 인간의 뇌처럼 기계를 작동시킬 수 있을 것이라고 주장

* 프랑크 로젠블라트가 이 이론을 실제 장치로 구현
* 이를 **퍼셉트론**이라고함

  * $x_0, x_1, x_2, ..., x_n$의 각각 입력 값을 받아서
  * $x_o * x_0 + b, x_1 * x_1 + b, x_n * x_n + b$각각을 가중치와 곱해서 (bias도 더함)
  * ($x_i * w_i + b$)이 특정 임계값을 넘으면 1 그렇지 않으면 0 과 같은 알고리즘으로 구성됨
  * 실제 결과의 오차가 생기면, 가중치를 업데이트 함

    * 실제 결과와의 오차를 기반으로 가중치 업데이트를 할 수 있는 구조로 인해, 최초로 **학습**이 가능하게 됨


### 퍼셉트론의 이슈 1

* 조건을 만족하는 선이 여러개가 나와서, 최적의 선을 구하기 어려움
* 아달라인(Adaline)

  * 경사하강법을 도입하여 가중치를 업데이트 시키므로써, 퍼셉트론의 이슈를 보완

### 퍼셉트론의 이슈 2

* 하나의 레이어를 가지고, 0과 1을 출력하는 퍼셉트론 구조는 직선형태의 구분선을 만들 수 있으며,
* 직선 형태의 구분선으로는 XOR 문제를 풀 수 없음

### XOR 이슈 해결

* NAND, OR, AND 게이틀 조합하면 XOR 게이트를 만들 수 있음
* 이 XOR 게이트로 컴퓨터 CPU가 덧셈을 할 수 있게 됨

* 다층 퍼셉트론

  * XOR 문제는 은닉층을 만들어 해결할 수 있었음
  * 즉 여러 퍼센트론 레이러를 두고, 해결함

* 입력과 출력이 아닌 숨겨진 은닉층을 만들면, 복잡한 문제를 풀 수 있음
* 은닉층이 2개 이상인 신경망을 Deep Neural Network, DNN 이라고 함
