# 하이퍼 파라미터 vs 모델 파라미터

### 모델 파라미터

* 모델 내부의 설정 값으로, 데이터 학습에 의해 모델 내부에서 값이 자동으로 정해짐
* 딥러닝에서는 Network Weight Parameter 라고도 불림

### 하이퍼 파라미터

* 모델 생성시, 사용자가 직접 입력하는 변수
* 예를 들어, learning rate는 모델 생성시 사용자가 입력해야 하는 변수이므로, 하이퍼 파라미터 임

### 차이

* 머신러닝에서, 모델의 파라미터는 학습 과정에서 결정되는 변수라고 이해하고,
* 하이퍼 파라미터는 모델 생성시 사용자가 직접 입력해줘야 하는 변수라고 구별하면 이해하기 쉬움

### 하이퍼 파라미터 튜닝이란?

* 사용자가 입력하는 변수값을 과연 어떤 값으로 넣어줘야 성능이 최적화될지를 찾는 작업

## 참고 : 하이퍼 파라미터 튜닝을 위해 참고할만한 사안들

* 하이퍼 파마리터 튜닝을 위해, 다양한 실험을 진행해야 함

  * 초간단 모델 구축에서 부터, 조금씩 모델을 개선해가는 방식으로 성능 튜닝을 하는 것이 일반적임

* 다양한 실험 결과와 그 때의 설정 값을 관리해주는 프레임워크가 잇음

  * 예 : MLFlow, WanDB 등
