# 텐서 

## 텐서 정리

```
딥러닝은 텐서로 데이터를 다루므로 텐서 관련 용어를 정리하기로 함
```

* 텐서는 데이터는 개수와 형태에 따라 스칼라(scalar), 벡터(vector), 행렬(matrix), 텐서(tensor)로 분류할 수 있음

```
4차원 이상은 그림으로 표기할 수 없으므로, 보통 3차원과 동일하게 표기함
```

## 텐서를 정의하는 용어

* 축(axis)의 갯수(rank): 1D 텐서(벡터)는 1개의 축이 있고, 2D 텐서(행렬)은 2개의 축이 있음
    * pytorch 에서는 텐서 객체.dim()으로 축의 갯수를 확인할 수 있음
    * 축의 갯수를 다른말로 rank(랭크)라고도 함

* 크기(shape 또는 size) : 텐서의 **각 축(axis)에 따라, 얼마나 많은 차원이 있는지**를 나타내는 값으로, 파이썬의 튜플 구조로 표현
* 용어가 헷깔리므로, 가능한 shape, size를 기반으로 실제 구조를 확인하는 편이 좋음
    * 예: 2D 텐서(matrix)는 축(axis)가 두개 있다라는 것을 의미
    * 예: 3D 벡터(또는 3 dimension, 또는 3차원 백터)라는 의미는 벡터 이므로 축(axis)는 한개 있지만, 원소를 가지고 있다는 의미임
    
## 스칼라 (OD 텐서)

* 하나의 숫자만 담고 있는 텐서를 의미함
* 스칼라, 스칼라 텐서, 0차원 텐서, 또는 0D 텐서라고 부름
* 스칼라는 다음과 같이 표기함 $x \epsilon \mathbb{R}$

    * $\mathbb{R}$을 실수 집합이라고 할 때, 스칼라 $x$는 다음과 같이 표현함

        * 참고 : $\mathbb{R}$는 칠판체라고 하며, 수학 기호를 칠판에 굵게 표시하려다, 내부에 칠하지 않는 경우가 많아져서, 굳어진 심볼임
        * $\mathbb{R}$는 실수를 나타내는 기호

    * 참고 : $\epsilon$ 기호 - 특정 원소가 집합에 포함되어 있음을 의미함
        
        * $A = {1, 2, 3, 4}$라고 할 때, 2는 A의 원소로 포함되어 있다라고 표현할 때, $2 \epsilon A$라고 표기

## 벡터 (1D 텐서)

* 숫자 여러개가 특정 순서대로 모여 있는 것을 의미함 (배열이라고 이해하면 됨)

    * $x  \epsilon   \mathbb{R}^n$, (n, )
    * 하나의 벡터를 이루는 데이터의 갯수가 $n$개이면, n-차원 벡터(n-dimensional vector, $nD$ vector) 라고 함
    * 벡터는 열 벡터, 행 벡터 둘 다를 가리키지만, 열 벡터로 표현하는 것이 일반적


## 행렬 (2D 텐서)

* 가로, 세로 두 축을 가지는 2차원 텐서
* 다음 x 데이터에서는 $x_{ij}$는 $(i,j)$ 번째 데이터를 의미함
    
    * 첫 번째 숫자는 행축 (row/ axis), 두 번째 숫자는 열축(column axis)을 의미함
    * $\mathbb{R}^{\text{행의갯수} \times \text{열의갯수}}$ 로 표시함
        
        * 행의 갯수는 행의 차원, 열의 갯수도 열의 차원이라고도 이야기 할 수 있음
            * $\mathbb{R}^{n \times 2}$ , (n, 2)

        * 열의 갯수는 머신러닝의 feature 갯수로 볼 수 있음
        * (n,2)와 같이 표기시, 읽을 때에는 'n 바이 2'라고 이야기함

* 벡터와 스칼라를 다음과 같이 표기할 수도 있음

* $\mathbb{R}^{n \times 1}$ , (n,)

## 텐서

* 3차원 이상의 행렬의 의미

* 이 중에서 3D 텐서는,

    * 크기가 동일한 행렬을 여러개 묶어놓는 데이터
    * $\mathbb{R}^{k \times n \times m}$ , (k, n, m)
    * 상당이 헷깔리므로, 높이 너비, 깊이 순서를 기억하는 편이 좋음
